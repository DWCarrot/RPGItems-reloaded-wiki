## Description

## Properties

* chance

  * Type: int
  * **Required**

  Chance of triggering this power, 1/n

* cost

  * Type: int
  * Default: 0

  Durability cost of the power.

* count

  * Type: int
  * Default: 1

  Times to run the command

* command

  * Type: String

  Command to be executed

* desc

  * Type: String

  Description in display text
