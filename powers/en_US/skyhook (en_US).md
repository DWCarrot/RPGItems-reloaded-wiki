## Description

## Properties

* cost

  * Type: int
  * Default: 0

  Durability cost of the power.

* railMaterial

  * Type: Material
  * **Required**

  Material that can hooks on

* cooldown

  * Type: long
  * Default: 20

  Cooldown of the power, in ticks.

* hookDistance

  * Type: int
  * Default: 10

  Maximum distance

* hookingTickCost

  * Type: int
  * Default: 0

  Cost per tick hooking
